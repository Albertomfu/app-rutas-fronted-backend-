Sobre ExtreRutas

Una guía interactiva para descubrir las mejores rutas senderistas de Extremadura.

🌿 ¿Qué es ExtreRutas?

ExtreRutas es una aplicación web dedicada a recopilar y mostrar rutas de senderismo por toda Extremadura, organizadas por zonas y comarcas, pensada tanto para aficionados como para amantes de la naturaleza.

🗺️ ¿Qué encontrarás aquí?

  • Rutas clasificadas por zonas de Extremadura
  • Mapas interactivos para explorar el territorio
  • Información básica de cada ruta
  • Posibilidad de guardar rutas favoritas

🥾 ¿Para quién está pensada?

Esta web está pensada para personas que disfrutan del senderismo, escapadas rurales, turismo de naturaleza y para quienes quieren descubrir rincones menos conocidos de Extremadura.

⚙️ Tecnologías utilizadas

El proyecto está desarrollado con Angular y tecnologías web modernas, siguiendo una estructura modular y escalable.
